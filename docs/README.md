# ドキュメント構成

このディレクトリには学習プロジェクトに関連するドキュメントが整理されています。

## ディレクトリ構造

```
docs/
├── README.md                    # このファイル（ドキュメント構成の説明）
├── progress/                    # 学習進捗管理
│   └── LEARNING_PROGRESS.md     # 現在の学習状況まとめ
└── learning-materials/          # 学習資料
    ├── learning-flow.adoc       # 全体的な学習フローの概要
    ├── phase2-react-basics.adoc # Reactコンポーネントの基礎とJSX
    ├── phase2-react-state.adoc  # React状態管理
    ├── phase3-nextjs-fundamentals.adoc # Next.js基本機能
    ├── phase3-nextjs-api.adoc   # Next.js API開発
    ├── phase4-blog-development.adoc # ブログアプリ開発
    └── phase5-7-deployment.adoc # デプロイメント

## ファイルの説明

### progress/
- **LEARNING_PROGRESS.md**: 実装済みコンポーネントと学習進捗の詳細な記録

### learning-materials/
- 各フェーズの詳細な学習資料（AsciiDoc形式）
- 段階的な学習フローに基づいた教材

## 更新頻度
- **progress/**: 新しいコンポーネント実装や学習完了時に更新
- **learning-materials/**: 学習フェーズの進行に応じて参照・更新
